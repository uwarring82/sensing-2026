# Sensing 2026

An interactive companion to the **Quantum Hardware** lectures, University of Freiburg —
extreme and quantum sensing, from human perception to the standard quantum limit.

**▶ Open it: https://uwarring82.github.io/sensing-2026/**

Three rooms:
- **Lecture** — the two 90-minute interludes, read as one story.
- **Workshop** — interactive widgets for each idea, plus the spaced-week exercises.
- **Library** — a thin reference card per concept (definition, equations, the one common misuse, references).

Every page is a single, self-contained HTML file that works offline with no tracking.
By U. Warring (group of T. Schaetz). Sources are linked, not rehosted; this repo holds only
the public student-facing hub — the full course source is maintained in a separate private repo.
